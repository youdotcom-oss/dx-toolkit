## Configuration Options

**Hosted MCP** (Recommended):
- OpenAI-managed connection
- Simpler configuration
- Lower latency
- Automatic retries

**Streamable HTTP**:
- Self-managed connection
- Full control over networking
- Custom infrastructure
- Better for development/testing
